## 这个项目是做什么的

将一些经常使用的工具（操作步骤很少）做成集合。

特色：
1. all in web
2. sugar gui
3. easy

## 调试本项目

``` bash
git clone https://github.com/cocaine-coder/wang-tools.git

npm install     # yarn 
npm run dev     # yarn dev

npm run test    # yarn test
npm run build   # yarn build

```
